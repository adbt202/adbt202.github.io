Bootcamp 2020: Ball Game

My first programming project, concluding the 2 week Programming Bootcamp at City (2020). During the last few days I have decided to make a simple ball game using some key topics learnt during the bootcamp which include loops, classes, random, map and also some sound files. The game consists of 3 ellipses which have different properties. The player uses the mouse to control the blue ellipse and the objective is to eat the smaller yellow circles(Food) while dodging the red ball which gets faster as the score increases. Once the red circle comes in contact with the blue ball the game stops and the screen displays "GAMEOVER" followed by your score and also "Click to play again". I also decided to change the background colour using map which allowed me to change it using mouse value.

Check out the code on Github: https://github.com/adbt202/adbt202.git
